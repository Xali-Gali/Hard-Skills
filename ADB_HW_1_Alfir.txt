ADB HW_1

Сценарий напишите в .txt файл.
.txt сценарий и .log файл прииложения todolist выгружайте на GitHub.
Ссылкку на гитхаб присылайте в лс.

 1. Отобразить подключённый девайс в консоли.
 Прописать adb devices(должно появиться номер девайса)
 2. Вывести адрес приложения todolist в системе Android
 adb shell "pm list packages -f | grep todolist"
 3. Установить .apk файл приложениия todolist на телефон с компьютера через  ADB
 adb install -t путь к .apk
 4. Сделать скриншот запущенного приложения todolist и сразу скопировать на компьютер в одной команде.
adb shell screencap /storage/emulated/0/DCIM/Screenshots/1213.png && adb pull /storage/emulated/0/DCIM/Screenshots/1213.png "C:\Users\ALFIR-PC\platform-tools\1213.png"
 5. Вывести в консоль логи приложения todolist
 adb shell "logcat|grep -rnw 'com.android.todolist'"
 6. Скопировать логи приложения todolist на компьютер.
 adb shell "logcat|grep -rnw 'com.android.todolist'" > log.txt 
 7. Удалить приложение todolist с телефона через ADB
adb uninstall com.android.todolist
